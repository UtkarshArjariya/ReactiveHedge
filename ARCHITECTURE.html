ReactiveHedge

A Uniswap v4 hook that hedges impermanent loss before the arbitrage lands. v4 hooks only run when the pool is touched — but IL accrues when the price moves elsewhere. A Reactive Smart Contract watches that other venue and fires a cross-chain callback into the hook within seconds.

Non-negotiable guard rails

Address flags

getHookPermissions() must match the mined salt. Change the active callbacks in both the hook and the HookMiner call, or deployment reverts.

Callback auth

Validate proxy + RVM id on every callback. Reactive injects the RVM id as the first address arg — check it, or anyone can spoof a hedge.

Dual instance

subscribe under if(!vm); react under vmOnly. The RSC runs as two instances; mixing them silently breaks subscriptions.

Same realm

Never mix mainnet and testnet. Origin and destination must both be testnet here: Unichain Sepolia, Base Sepolia, Lasna.

Fund the RSC

An underfunded RSC gets blocklisted. Deposit REACT after deploy; callbacks ≥ 100k gas, ≤ 900k RVM gas.

Reorg aware

Callbacks can fire before origin finality. Keep hedge updates idempotent / reversible; don't assume the origin event is final.
